# GifTastic

## Built With
* AJAX
* API
* jQuery
* JavaScript
* HTML
* CSS
* Bootstrap 

### Overview
Use the GIPHY API to make a dynamic web page that populates with gifs of your choice. First the webiste must call the GIPHY API. Then it uses JavaScript and jQuery to change the HTML of the site.
